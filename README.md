**tl;dr**

* [Fastify](https://github.com/fastify/fastify) is, fast and low overhead web framework for Node.js
* This package shows how fast it is comparatively.

**Installing**

```
npm i -g fastify-benchmarks
```

**Usage**

```
fastify-benchmark # Hit [ENTER]
```
